Description : Ce module vous permet de générer instantanément et à tout moment les documents comptables suivants à partir des informations enregistrés en comptabilité dans Dolibarr :
  • Compte de résultat
  • Bilan
  • Balance auxiliaire clients
  • Balance auxiliaire fournisseurs
La présentation et les numéros de comptes utilisés sont ceux de la comptabilité  française.
Les documents sont générés de date à date. La date est paramétrable, ce qui permet de sortir les documents pour des périodes spécifiques, sur des exercices décalés ou sur plusieurs exercices.
Version : 1.4

Pré-requis :
  • Version minimum supportée de Dolibarr : 12.0.x
  • Version maximum : 15.0.x
  • Dolibarr : module du core "Comptabilité (partie double)" activé
  • Module Progiseize (gratuit) installé et activé - [GitHub]
  • Prérequis comptables et techniques :
    • Seules les écritures du grand livre sont prises en compte : les lignes à lier doivent être liées, et les journaux transférés en comptabilité
    • Un accès à la base de données est nécessaire pour les points suivants. De multiples tables de la base de données sont concernées.
      • Les codes comptables doivent être homogènes et cohérents.
      • Les codes comptables auxiliaires doivent être homogènes et cohérents.
      • Il ne doit pas y avoir de doublon de code comptable.
Nous conseillons fortement, si disponible, de repartir du dernier FEC (Fichier des Ecritures Comptables) validé par le comptable afin de :
  • Se baser sur les codes utilisés par le comptable
  • Lors de la première installation : réintégrer par différentiel (entre les informations du grand livre Dolibarr et le FEC validé) une écriture d'OD afin de de partir sur un bilan validé en début d'exercice
  • Pour les exercices comptables suivants : le comptable pourra exporter le FEC de Dolibarr, travailler sur son logiciel comptable, puis fournir directement une liste des OD qu'il a réalisé et qui sont à réintégrer dans Dolibarr

    • Ces opérations demandent un savoir-faire technique et comptable avancé, ainsi que des opérations en base de données.
